# ModernWorldV2 - Back-end

JS (Nest.js) -> JAVA (Spring Boot) 전환을 위한 레포지토리입니다.

실무 수준의 이슈 관리와 Git 전략을 적용하는 것을 목표로 합니다.

---

## 📌 기존 버전과의 차별점

- Junit을 활용한 테스트 구현
- Thymeleaf를 이용한 Admin Page 구현
